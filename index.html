@zenebyte/unstyled

Framework-agnostic Web Components

An unstyled component library built with Lit. Every component is a standard Custom Element — usable in React, Vue, Svelte, Angular, or plain HTML without modification. Ships zero visual styles; bring your own via class attributes or ::part() selectors.
